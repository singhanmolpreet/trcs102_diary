# Day 1 - Introduction to Python

## Date

Day 1 of ML Training

## Objective

To understand the fundamentals of Python programming and become familiar with basic data structures used in Machine Learning.

## Topics Covered

### 1. Python Basics

* Introduction to Python
* Variables and data types
* Input and output functions
* Basic operators and expressions

### 2. Lists

* Creating lists
* Accessing list elements
* List slicing
* Common list methods such as `append()`, `remove()`, and `sort()`

### 3. Strings

* String creation and manipulation
* Indexing and slicing
* String methods
* String formatting

### 4. Tuples

* Creating tuples
* Accessing tuple elements
* Difference between lists and tuples
* Advantages of immutable data structures

## Activities Performed

* Wrote basic Python programs.
* Practiced list operations and string manipulation.
* Implemented examples using tuples.
* Solved simple programming exercises.

## Key Learnings

* Python is a simple and powerful programming language.
* Lists are mutable and useful for storing collections of data.
* Strings provide various methods for text processing.
* Tuples are immutable and suitable for fixed data.

## Summary

The first day focused on building a foundation in Python programming. Understanding these basic concepts is essential for learning data analysis and Machine Learning in the upcoming sessions.
