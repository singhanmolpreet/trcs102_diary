# Day 3 - Classes in Python

## Date

Day 3 of ML Training

## Objective

To understand the concept of classes and objects in Python and learn how they support object-oriented programming.

## Topics Covered

### 1. Classes

* Introduction to classes
* Creating a class
* Class attributes and instance attributes
* The `__init__()` constructor
* The `self` keyword

### 2. Objects

* Creating objects from a class
* Accessing attributes
* Calling methods
* Creating multiple objects

## Activities Performed

* Created custom classes.
* Defined constructors using `__init__()`.
* Created objects and accessed their attributes and methods.
* Practiced writing simple class-based programs.

## Key Learnings

* Classes act as blueprints for creating objects.
* Objects store data and perform actions through methods.
* Constructors initialize object attributes automatically.
* Object-oriented programming improves code organization and reusability.

## Summary

Day 3 introduced the fundamentals of classes and objects, providing the foundation for object-oriented programming in Python.
