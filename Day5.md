# Day 5 - NumPy

## Date

Day 5 of ML Training

## Objective

To learn the fundamentals of NumPy for efficient numerical computing and array operations.

## Topics Covered

### 1. Introduction to NumPy

* Installing and importing NumPy
* NumPy arrays
* Advantages over Python lists

### 2. Array Operations

* Creating arrays
* Indexing and slicing
* Reshaping arrays
* Mathematical operations

### 3. Useful Functions

* `zeros()`
* `ones()`
* `arange()`
* `linspace()`
* `reshape()`
* Aggregate functions such as `sum()`, `mean()`, and `max()`

## Activities Performed

* Created one-dimensional and multi-dimensional arrays.
* Performed arithmetic and statistical operations.
* Practiced array indexing, slicing, and reshaping.
* Solved numerical computing exercises.

## Key Learnings

* NumPy provides efficient storage and computation for numerical data.
* Array operations are faster than traditional Python lists.
* Vectorized operations simplify mathematical computations.

## Summary

Day 5 introduced NumPy, an essential library for scientific computing and the foundation for data analysis and Machine Learning.
