# Day 16 - Linear Regression

## Date

Day 16 of ML Training

## Objective

To understand the fundamentals of Linear Regression and learn how it is used to predict continuous numerical values.

## Topics Covered

### 1. Introduction to Linear Regression

* What is Regression?
* Difference between Regression and Classification
* Applications of Linear Regression

### 2. Linear Regression Model

* Simple Linear Regression
* Multiple Linear Regression
* Best Fit Line
* Dependent and Independent Variables

### 3. Model Training

* Training a Linear Regression model
* Making predictions
* Understanding model coefficients and intercept

### 4. Model Evaluation

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score (Coefficient of Determination)

## Activities Performed

* Prepared the dataset for regression.
* Trained a Linear Regression model.
* Predicted target values using the trained model.
* Evaluated model performance using regression metrics.
* Interpreted the regression coefficients.

## Key Learnings

* Linear Regression predicts continuous values by finding the best-fitting linear relationship between features and the target variable.
* Model evaluation metrics help measure prediction accuracy.
* R² Score indicates how well the model explains the variance in the target variable.
* Proper preprocessing improves regression model performance.

## Summary

Day 16 introduced Linear Regression, one of the most fundamental supervised Machine Learning algorithms. The session covered model training, prediction, and evaluation using common regression metrics.
