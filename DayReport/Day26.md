# Day 26 - Introduction to OpenCV

## Date

Day 26 of ML Training

## Objective

To learn the basics of OpenCV and perform fundamental image processing operations using Python.

## Topics Covered

### 1. Introduction to OpenCV

* What is OpenCV?
* Applications of OpenCV
* Installing and importing OpenCV

### 2. Basic Image Operations

* Reading an image using `cv2.imread()`
* Displaying an image
* Converting an image to grayscale
* Saving an image using `cv2.imwrite()`

### 3. Image Properties

* Image shape
* Height
* Width
* Number of channels

### 4. Working with Images

* Understanding pixel representation
* Accessing image dimensions
* Basic image inspection

## Activities Performed

* Loaded images using OpenCV.
* Displayed and saved images.
* Converted color images to grayscale.
* Retrieved image dimensions and channel information.
* Explored basic image properties.

## Key Learnings

* OpenCV is a powerful library for computer vision and image processing.
* Images are represented as NumPy arrays.
* Grayscale conversion simplifies many image processing tasks.
* Understanding image dimensions and channels is essential before performing advanced image operations.

## Summary

Day 26 introduced the fundamentals of **OpenCV**, focusing on reading, displaying, converting, and saving images. Basic image properties such as shape, height, width, and channels were explored, laying the foundation for more advanced computer vision applications.
