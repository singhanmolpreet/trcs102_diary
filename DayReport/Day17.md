# Day 17 - Polynomial Regression and Logistic Regression

## Date

Day 17 of ML Training

## Objective

To understand Polynomial Regression for modeling non-linear relationships and Logistic Regression for solving classification problems.

## Topics Covered

### 1. Polynomial Regression

* Introduction to Polynomial Regression
* Difference between Linear and Polynomial Regression
* Polynomial Features
* Selecting the degree of the polynomial
* Predicting non-linear relationships

### 2. Logistic Regression

* Introduction to Logistic Regression
* Regression vs Classification
* Binary Classification
* Sigmoid Function
* Decision Boundary
* Predicting class labels and probabilities

### 3. Model Evaluation

* Accuracy Score
* Confusion Matrix
* Precision
* Recall
* F1-Score

## Activities Performed

* Generated polynomial features from input data.
* Trained a Polynomial Regression model.
* Built a Logistic Regression classifier.
* Evaluated classification performance using appropriate metrics.
* Compared Linear Regression, Polynomial Regression, and Logistic Regression.

## Key Learnings

* Polynomial Regression extends Linear Regression by modeling non-linear relationships.
* Logistic Regression is a supervised classification algorithm despite its name.
* The sigmoid function converts predictions into probabilities between 0 and 1.
* Different evaluation metrics are required for regression and classification models.

## Summary

Day 17 covered two important supervised learning algorithms: **Polynomial Regression**, which captures non-linear patterns in data, and **Logistic Regression**, which is widely used for binary classification problems. The session also introduced common classification evaluation metrics.
