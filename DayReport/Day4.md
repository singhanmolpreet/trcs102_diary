# Day 4 - Object-Oriented Programming (OOP) Concepts

## Date

Day 4 of ML Training

## Objective

To understand the four fundamental principles of Object-Oriented Programming (OOP) in Python.

## Topics Covered

### 1. Encapsulation

* Bundling data and methods into a single unit
* Access modifiers
* Protecting object data

### 2. Abstraction

* Hiding implementation details
* Exposing only essential functionality
* Introduction to abstract classes

### 3. Inheritance

* Reusing code from existing classes
* Parent and child classes
* Method overriding

### 4. Polymorphism

* Same interface with different implementations
* Method overriding
* Duck typing in Python

## Activities Performed

* Implemented inheritance using parent and child classes.
* Practiced method overriding.
* Explored encapsulation through class attributes.
* Created examples demonstrating polymorphism.

## Key Learnings

* OOP improves code modularity and maintainability.
* Encapsulation enhances data security.
* Inheritance promotes code reuse.
* Polymorphism allows flexible and extensible program design.

## Summary

Day 4 focused on the core principles of object-oriented programming, which are widely used in software development and Machine Learning applications.
