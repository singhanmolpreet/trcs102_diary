# Day 2 - Sets and Dictionaries

## Date

Day 2 of ML Training

## Objective

To learn Python data structures that help organize, store, and retrieve data efficiently.

## Topics Covered

### 1. Sets

* Introduction to sets
* Creating sets
* Adding and removing elements
* Set operations:

  * Union
  * Intersection
  * Difference
  * Symmetric Difference

### 2. Dictionaries

* Introduction to key-value pairs
* Creating dictionaries
* Accessing dictionary values
* Adding and updating entries
* Dictionary methods and operations

## Activities Performed

* Created sets and performed set operations.
* Implemented examples using dictionaries.
* Practiced storing and retrieving data using key-value pairs.
* Solved programming exercises involving sets and dictionaries.

## Key Learnings

* Sets store unique values and are useful for removing duplicates.
* Set operations simplify data comparison tasks.
* Dictionaries provide fast access to data using keys.
* Dictionaries are widely used for representing structured information.

## Summary

The second day introduced sets and dictionaries, two powerful Python data structures frequently used in data processing and Machine Learning applications. These concepts will help in managing and manipulating datasets efficiently.
