# Day 27 - Image Transformations using OpenCV

## Date

Day 27 of ML Training

## Objective

To learn basic image transformation techniques in OpenCV, including resizing, rotating, flipping, and cropping images.

## Topics Covered

### 1. Image Resizing

* Resizing images using `cv2.resize()`
* Changing image dimensions
* Maintaining aspect ratio

### 2. Image Rotation

* Rotating images by a specified angle
* Understanding the rotation matrix
* Using `cv2.getRotationMatrix2D()`
* Applying rotation with `cv2.warpAffine()`

### 3. Image Flipping

* Horizontal flip
* Vertical flip
* Both horizontal and vertical flip
* Using `cv2.flip()`

### 4. Image Cropping

* Cropping images using array slicing
* Selecting a Region of Interest (ROI)
* Saving the cropped image

## Activities Performed

* Resized images to different dimensions.
* Rotated images at various angles.
* Applied horizontal and vertical flipping.
* Cropped specific regions from images.
* Saved the transformed images for comparison.

## Key Learnings

* Image resizing is useful for preparing images for Machine Learning and Deep Learning models.
* Rotation helps create augmented datasets and correct image orientation.
* Flipping images is a common data augmentation technique used in computer vision.
* Cropping allows extraction of the Region of Interest (ROI) from an image for further analysis.

## Summary

Day 27 focused on essential image transformation techniques using **OpenCV**. The session covered resizing, rotating, flipping, and cropping images, providing practical skills that are widely used in image preprocessing, data augmentation, and computer vision applications.
