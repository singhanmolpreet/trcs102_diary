# Day 19 - Support Vector Machine (SVM) and K-Nearest Neighbors (KNN)

## Date

Day 19 of ML Training

## Objective

To understand two popular supervised Machine Learning algorithms: Support Vector Machine (SVM) and K-Nearest Neighbors (KNN) for classification tasks.

## Topics Covered

### 1. Support Vector Machine (SVM)

* Introduction to SVM
* Hyperplane and Decision Boundary
* Support Vectors
* Linear and Non-Linear SVM
* Kernel Functions

  * Linear Kernel
  * Polynomial Kernel
  * RBF (Radial Basis Function) Kernel

### 2. K-Nearest Neighbors (KNN)

* Introduction to KNN
* Distance-based classification
* Choosing the value of **K**
* Euclidean Distance
* Advantages and limitations of KNN

### 3. Model Evaluation

* Accuracy Score
* Confusion Matrix
* Classification Report

## Activities Performed

* Trained an SVM classifier.
* Built a KNN classification model.
* Compared predictions from both algorithms.
* Evaluated model performance using classification metrics.

## Key Learnings

* SVM separates classes by finding the optimal decision boundary.
* Kernel functions enable SVM to solve non-linear classification problems.
* KNN classifies a data point based on the majority class of its nearest neighbors.
* The choice of **K** significantly affects KNN performance.

## Summary

Day 19 introduced **Support Vector Machine (SVM)** and **K-Nearest Neighbors (KNN)**, two widely used classification algorithms. Their working principles, applications, and evaluation techniques were explored through practical implementation.
