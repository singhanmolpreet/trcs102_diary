# Day 25 - Introduction to Convolutional Neural Networks (CNN)

## Date

Day 25 of ML Training

## Objective

To understand the fundamentals of Convolutional Neural Networks (CNNs) and build a basic image classification model to distinguish between cats and dogs.

## Topics Covered

### 1. Introduction to CNN

* What is a Convolutional Neural Network?
* Difference between ANN and CNN
* Applications of CNN

  * Image Classification
  * Object Detection
  * Face Recognition
  * Medical Image Analysis

### 2. CNN Architecture

* Convolution Layer
* Activation Function (ReLU)
* Pooling Layer (Max Pooling)
* Flatten Layer
* Fully Connected Layer
* Output Layer

### 3. Image Classification

* Preparing image datasets
* Training a CNN model
* Binary classification (Cat vs Dog)
* Model prediction

## Activities Performed

* Learned the architecture of a CNN.
* Built a basic CNN model for binary image classification.
* Trained the model on a Cats vs Dogs dataset.
* Evaluated the model by predicting whether an image contained a cat or a dog.

## Key Learnings

* CNNs automatically extract important features from images.
* Convolution and pooling layers reduce the need for manual feature engineering.
* CNNs are highly effective for computer vision tasks.
* Deep learning enables accurate image classification with sufficient training data.

## Summary

Day 25 introduced **Convolutional Neural Networks (CNNs)** and demonstrated how they can be used for image classification. A basic CNN model was built to classify images of cats and dogs, providing an introduction to deep learning for computer vision.
